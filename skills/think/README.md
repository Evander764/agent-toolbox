# Think

Short invocation:

```text
$think <goal or question>
```

`think` is a second-brain Skill for adversarial multi-agent reasoning. It uses
first principles, entropy/system-failure analysis, Munger-style mental models,
falsification, and smallest-useful-experiment design.

The Skill writes proposed improvements to `evolution-proposal.md` for review
instead of silently rewriting itself.
